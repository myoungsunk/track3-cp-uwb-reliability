# Geometric Validation Execution Conditions

This bundle was generated by rerunning the geometric-target workflow and saving each stage into a dedicated directory.

## Reproduction Command

```matlab
matlab -batch "outputs = run_geometric_story_bundle;"
```

## Key Notes

- The full baseline stage uses all 6 cases: `CP_caseA/B/C`, `LP_caseA/B/C`.
- The CP7 stages use `CP_caseB` and `CP_caseC` only, because the 6 channel-resolved CP7 features are available only there.
- The reviewer stage stores both `material` and `geometric` outputs because the downstream priority script expects the material dataset for its shared-correlation block; the final report still uses geometric outputs only.
- Therefore, `Stage 01` and `Stage 03-05` are based on different sample universes and their AUC values should not be compared as if they were from the same dataset.
- The hard-case subset is defined as baseline OOF confidence in `[0.4, 0.6]`.
